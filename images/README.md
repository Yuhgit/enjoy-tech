## Figmaからのエクスポート方法

1. Figmaで該当する要素を選択
2. 右クリック → 「エクスポート」を選択
3. 適切な形式（PNG/SVG）とサイズを選択
4. 上記のファイル名で保存
5. `images`フォルダに配置
